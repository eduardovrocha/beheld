# beheld-engine sha256 hashes

Reproducible-build hashes (F5.7.1) of the `beheld-engine` PyInstaller binary
for each released version. Verifiers and snapshot consumers can use these to
check a bundle's `engine_version_hash` against the official release table.

Hashes are generated by `scripts/build.sh` and validated by
`scripts/verify-reproducible.sh` (which runs two builds back-to-back and
fails if they diverge).
